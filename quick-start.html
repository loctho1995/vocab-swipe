📚 Vocab Swipe

Học từ vựng Anh-Việt với Sources Management

  • 📁 Quản lý nhiều bộ từ vựng (Sources)
  • 📥 Import từ file JSON có cấu trúc
  • 📊 Theo dõi tiến độ học tập chi tiết
  • 👆 Swipe gesture: Trái = Đã học, Phải = Bỏ qua
  • 🔊 Phát âm với audio tùy chỉnh
  • 📖 Xem lịch sử từ đã học
  • 💾 Lưu trữ local - không cần internet

🚀 Bắt đầu nhanh:

1. Mở ứng dụng

2. Click "Import từ vựng"

3. Dán JSON từ file trong folder sources/

4. File sẽ tự động tải về - lưu lại vào sources/

5. Bắt đầu học!
